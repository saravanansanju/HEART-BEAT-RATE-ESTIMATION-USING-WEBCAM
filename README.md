# Webcam Pulse Detector

This project detects pulse from webcam video using simple face detection and signal processing.

Quick start

1. Install requirements (create a virtual environment first):

```powershell
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Run the main script:

```powershell
python get_pulse.py
```

Notes

- Add a `requirements.txt` if you want reproducible installs.
- Create a GitHub repo and push this project (see instructions in issues or run the helper commands).
